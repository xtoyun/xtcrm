## [0.1] — 2025-03

### 新增

- 项目初始发布
- ThinkPHP 6.1 后端框架
- Vue 3 + Vite + Ant Design Vue 4 前端
- CRM 核心模块：客户、线索、报价、订单、合同
- 多租户架构（store_id 隔离）
- 五层 AI 架构：事件流 → Service → 洞察引擎 → 反馈闭环 → Agent 引擎
- 技能包（Skill）体系：内容引擎、代开发票、中文报价
- RBAC 权限管理（用户/角色/部门/菜单）
- 公海客户管理
- 知识库模块

## 版本命名规则

格式：`主版本.次版本`

- **主版本**：重大架构变更、不兼容的 API 变动
- **次版本**：功能新增、Bug 修复、文档更新

当前为 0.x 早期阶段，API 可能变动，不保证向后兼容。
